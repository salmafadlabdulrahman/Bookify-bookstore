## Overview
Bookify is a modern, full-featured e-commerce platform specializing in books, built with a MERN stack (MongoDB, Express, React, Node.js).
The application provides a seamless shopping experience with comprehensive product discovery, 
secure transactions, and an intuitive admin interface for store management.

### Features
Users should be able to:
- View the optimal layout for the app depending on their device's screen size.
- Clean modern layout.
- Intuitive dashboards to view books.
- Ability to register as an admin or owner or as a user.
- Contact us form to reach us through our email.
- Upload books, and make crud operations on books.
- Search and filter books by price.
- authentication and payment with stripe
